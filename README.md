# CampusWire

CampusWire is a Smart Campus Information & Community Platform built using PHP, MySQL, HTML, CSS, and JavaScript.

The platform centralizes campus communication through announcements, events, blogs, discussions, internships, and notifications using secure role-based access.

## Features

- Role-Based Authentication
- Campus News Portal
- Event Management System
- Blog Platform
- Discussion Board
- Internship Listings
- Notification System
- Admin Dashboard
